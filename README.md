# Portofolio Rizky Mema

Website portofolio pribadi yang modern, responsif, dan menarik secara visual, dibangun untuk menampilkan berbagai proyek, keahlian, dan pengalaman profesional saya.

## ✨ Fitur

- **UI/UX Modern**: Didesain dengan estetika yang rapi dan profesional menggunakan prinsip glassmorphism dan desain modern.
- **Desain Responsif**: Sepenuhnya responsif dan optimal untuk semua perangkat (ponsel, tablet, desktop).
- **Animasi Halus**: Terintegrasi dengan Framer Motion untuk transisi yang mulus, efek pengetikan (typing effect), dan mikro-interaksi.
- **Mode Gelap/Terang**: Beradaptasi secara otomatis terhadap preferensi pengguna dengan estetika yang dipoles dan kontras tinggi di kedua tema.
- **Creator Hub**: Bagian interaktif yang menampilkan sematan (embed) video media sosial dari TikTok dan Instagram.

## 🛠️ Teknologi yang Digunakan

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Bahasa**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Komponen UI**: [shadcn/ui](https://ui.shadcn.com/)
- **Animasi**: [Framer Motion](https://www.framer.com/motion/)
- **Ikon**: [Lucide React](https://lucide.dev/)

## 🚀 Memulai (Getting Started)

Ikuti langkah-langkah berikut untuk menjalankan proyek ini di komputer Anda secara lokal.

### Prasyarat

Pastikan Anda telah menginstal **Node.js** di sistem Anda.

### Instalasi

1. Clone repositori ini:
   ```bash
   git clone https://github.com/yourusername/portofolio-rizky.git
   ```
2. Masuk ke direktori proyek:
   ```bash
   cd "Portofolio Rizky"
   ```
3. Instal semua dependensi:
   ```bash
   npm install
   ```
4. Jalankan server pengembangan (development server):
   ```bash
   npm run dev
   ```
5. Buka browser Anda dan kunjungi `http://localhost:3000`.
